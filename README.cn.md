# 关于本项目

这是我的个人博客项目, 使用 hugo 搭建, 通过 `GitHub page` 功能进行静态页面的部署.

- main 分支是 `hugo` 的项目工程文件, 使用 [`bilberry hugo theme`](https://github.com/Lednerb/bilberry-hugo-theme) 主题
- page 分支是 `GitHub page` 的静态页面存放路径,从根路径 `/` 开始

# 关于 中英 内容的差别

中文内容与英文内容是由差别的! 

对于名人名言, 含有本土文化的内容等, `保持原样`是最佳策略.

对于部分文章我会用我那糟糕的英语和谷歌翻译将中文内容翻译成英文内容, 如果它能够被翻译的话.
对于另一部分文章我会使用不同的思路和表达方式来叙述同一件事.

所以不要用它来学英语!
除了 CN 内容下的 `Learning English` 系列, 这是为使用中文的人提供学习英语的学习笔记与一些经验,tips等.

如果你发现英文内容有错误,可以发邮件给我

或者可以前往讨论区的 `general` 分类留言.

或者你可以创建一个 `pull request` :)

如果你想报告一个检测到的bug, 请提 issue. (但是我很菜我不一定能解决
