# About This Project

This is my personal Blog project, build with `hugo` , deploying static page via `github page` , 

- branch `main` is the hugo project using [`bilberry hugo theme`](https://github.com/Lednerb/bilberry-hugo-theme) theme
- branch `page` is the static webpage project file of `Github page`, starting from the root path `/`

# About English & Chinese content

There are differences between Chinese content and English content, but it's almostly the same.

Only something like quote from fameous, or some content include native culture content, etc... `Keep it as it is` is better. 

For some articles I will use my poor English and Google Translate to translate the Chinese content into English, if it can be translated.
For another part of the article I will use different ideas and expressions to describe the same thing.

So Do Not Use It To Learning English!
Except `cn content` => `Learning English` series, that's for cn speakers give advices and notes, tips etc for Learning English.

You can email me if there's something wrong with English content through 

> <em class="fas fa-envelope"> Aeron.Atlantis@gmail.com </em>

Or leave the message on Discussion in general.

Or you can create a `pull request` :)

Please use the Issues section if you would like to report a defect or bug.
